🔑 APIキーとは？

APIキーは、GeminiのAIを使うための「合言葉」のようなものです。このアプリがAIと話せるようにするために必要です。

📋 取得方法

① ブラウザで「Google AI Studio」と検索する


② Googleアカウントでログインする


③ 左メニューの「Get API key」をタップ


④「APIキーを作成」ボタンを押す


⑤ 表示されたキーをコピーして、このアプリの入力欄に貼りつける
